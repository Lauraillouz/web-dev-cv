Memory card Game

Projet d'école réalisé en groupe :

Créer un jeu de cartes memory sans aucune consigne préalable.
Nous avons travaillé sur ce projet à 3. Mon rôle dans ce projet a été de coordonner et de distribuer les différentes tâches aux membres de l'équipe (travail en autonomie ou pari-programming).
J'étais aussi en charge de gérer les merge de notre branche develop à notre main sur GitHub.
4 grandes étapes dans le projet :
- Conceptualisation sur papier des différentes views, puis des différentes tâches/fonctions dont nous aurons besoin
- Distribution des tâches et mise ne place du projet à coder (phase de travail en autonomie)
- Pair-programming sur les fonctionnalités plus compliquées
- Présentation du projet devant la promo via des slides explicatives des étapes du projet et des fonctionnalités de l'application illustrées par le code

Technologies utilisées :

  • CSS3
  • React
  • GitHub (create branch / pull / merge / conflicts handling)

Web Design

Pretty Boys & Strong Girls

Projet artistique personnel :

Site créé en décembre 2020 pour mon projet artistique autour du féminisme. Le site comporte :
- un blog participatif (mise en forme gérée par mes soins),
- et une boutique où sont vendues mes oeuvres originales, des reproductions ainsi que des produits dérivés de la marque déposée.
Le site a un rôle de plateforme. L'idée était de concentrer sur un seul endroit les oeuvres, présenter la marque déposée en 2016, et lancer une communauté qui pourrait s'exprimer sur le blog.
Le but était donc de faire quelque chose d'assez neutre, aéré et simple afin de laisser place à la diversité d'informations.
Pour la réalisation, j'ai pu bénéficier des conseils de :
- Rackele Prencipe, conseillère en communication,
- et Hélène Windom, correctrice et rédactrice
La réalisation du projet web a été faite seule.

Technologies utilisées :

  • CSS3
  • Wordpress
